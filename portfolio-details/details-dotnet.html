Analysis

When I got accepted as an intern, I asked for some explanation on what I was going to use for my different missions, they told me that I would mostly be using "C# .NET" and "ASP .NET Core" since that was mostly what was used by them. I have always been a very big fan of object oriented programming, ever since I learned to work with Java in my first year of university, so you can imagine that learning this I was very happy. I took one this course so that I could have all the proper basics in order to work properly as a .NET Developper.

The course has brought back to me lots of concepts that I had forgotten, which have been more than helpful all along my internship. It has also talked about proper programming conventions in C# so that my code could be as clean as possible. The course took me a few weeks to finish up since I didn't have much time and it was quite a big read with some exercices, but it gave me confidence in using the language as well as helping me like it.
